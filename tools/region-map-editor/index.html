Aseprite Integration

i How it works: This generates a Lua script that Aseprite can run to apply your terrain color replacements and icon overlays directly to an .ase/.aseprite file.

Steps:
1. Enter the full path to your source .ase/.aseprite file below.
2. Click "Export Aseprite Script" to download a .lua file.
3. In Aseprite, go to File > Scripts > Open Scripts Folder and place the .lua file there.
4. Run it via File > Scripts > region_map_aseprite.

The script will open your source file, apply all terrain palette swaps (including route water recoloring), add icons on a new layer, and save the result. You can also run it headlessly:
aseprite -b --script region_map_aseprite.lua
